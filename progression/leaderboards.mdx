Leaderboards let you see how you stack up against the rest of the server. There are two leaderboard commands — one for general stats and one specifically for gambling.

## Message Leaderboard

**Command:** `.leaderboard` (or `.lb`, `.top`)

This is the main server leaderboard. Use the dropdown to switch between categories:

| Category | What It Ranks |
|---|---|
| **Messages** | Total messages sent in tracked channels |
| **Vanity** | Total vanity time (displayed as days/hours/minutes) |
| **Balance** | Current Pokécoin balance |
| **Claimed (Lifetime PC)** | Total Pokécoins claimed through the claim panel |
| **Tag** | Total server tag time |
| **Gambling** | Net gambling profit |

Each page shows **10 members** with navigation buttons to browse through the full rankings.

## Gambling Leaderboard

**Command:** `.gamblingleaderboard` (or `.glb`, `.gtop`)

A dedicated leaderboard focused entirely on gambling statistics. Use the dropdown to switch categories:

| Category | What It Ranks |
|---|---|
| **Most Wins** | Total gambles won |
| **Most PC Won** | Total Pokécoins earned from gambling |
| **Highest Win Rate** | Win percentage (min. 5 games) |
| **Most Games Played** | Total gambles played |
| **Net Profit** | Total earnings minus total losses |
| **Highest Volume** | Total Pokécoins wagered |
| **Best Win Streaks** | Longest consecutive win streak |
| **Most Losses** | Total gambles lost |
| **Most PC Lost** | Total Pokécoins lost |

<Note>
  **Highest Win Rate** requires at least 5 games played to qualify. This prevents someone with 1 win and 0 losses from appearing at the top.
</Note>

## Server Stats

**Command:** `.serverstats` (or `.stats`)

Not a leaderboard per se, but a useful overview of server-wide statistics:

- Total messages sent across the server
- Total unopened boxes across all users
- Total Pokécoins in circulation (combined balances)
- Total Pokécoins claimed (lifetime)
- Total boxes given out (lifetime)
- Total vanity time across all users
- Number of active vanity holders
- Users with over 100 messages
- Level distribution (how many members are at each level)

<Tip>
  Check the leaderboards regularly to see how you're progressing. It's motivating to watch yourself climb the ranks — and it helps you identify which areas to focus on.
</Tip>
